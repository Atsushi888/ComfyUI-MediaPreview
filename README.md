# ComfyUI-MediaPreview
All-in-one preview node for ComfyUI.

## Features
- Image / Video / Mask / media_path support
- No browser reload needed
- Works with async outputs
- Outputs image and mask tensors

## Installation
```
cd ComfyUI/custom_nodes
git clone https://github.com/Atsushi888/ComfyUI-MediaPreview.git
```
Restart ComfyUI.

## Quick Demo

Connect any of:
- image
- mask
- media_path

Preview updates automatically.
